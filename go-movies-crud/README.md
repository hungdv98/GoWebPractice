## ROUTES TO IMPLEMENT
| METHOD | ROUTE | FUNCTIONALITY |
| ------- | ----- | ------------- |
| *GET* | ```/movies``` | _Get All Movies_ |
| *GET* | ```/movies/{id}``` | _Get A Movie By ID_ |
| *POST* | ```/movies``` | _Create A Movie_ |
| *PUT* | ```/movies/{id}``` | _Update A Movie By ID_ |
| *DELETE* | ```/movies/{id}``` | _Delete A Movie By ID_ |
